# Developer Acceptance Criteria

The platform is ready for integration only when the developer demonstrates:

1. Parent account creation
2. Child profile creation and parent-child linking
3. Paid placement test checkout
4. Payment confirmation and entitlement creation
5. CEFR placement test submission
6. Level recommendation
7. Trial lesson unlock
8. Full course purchase
9. Automatic course enrolment
10. Learner dashboard
11. Lesson content access
12. Downloadable resource access tracking
13. Activity scoring and feedback
14. Adaptive next-step recommendation
15. Production/evidence submission
16. Progress report generation
17. Parent dashboard progress visibility
18. Teacher/class dashboard visibility
19. Certificate generation after completion
20. Admin overview of users, orders, payments, enrolments, progress, downloads and certificates
21. Protected resource/toolkit access
22. Legal pages exist and are CMS-editable
