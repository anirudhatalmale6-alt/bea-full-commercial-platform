# Benchmark Architecture Notes

## CoursesOnline-style homepage structure

The homepage uses a course-marketplace architecture: navigation by subject/level, a learner goal navigator, popular pathways, course cards, course levels, provider-style credibility sections and conversion-focused call-to-action buttons.

## Pearson/Cambridge-style placement journey

The placement journey begins before the curriculum. Candidates register, pay, take the adaptive test, then receive a score report and certificate. The engine uses calibrated items, adaptive routing and a theta-to-CEFR banding model.

## McGraw Hill-style curriculum depth

The curriculum library is organised by level, grade/learner stage, skill, modules, outcomes and programme pathways. This mirrors large publisher architecture without copying their content.

## SplashLearn/Starfall-style learning library

Activities are organised by level, skill, topic and type. The library includes games, lessons, worksheets, stories and speaking quests with positive reinforcement, visual practice loops, supports and challenge tasks.

## Non-copying statement

All course descriptions, activity prompts, item-bank questions, report statements and certificate wording in this package are original seed content created for the British English Academy platform.
