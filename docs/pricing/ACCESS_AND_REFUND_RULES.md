# Access and Refund Rules

## Access rules

### Public user

Can:

- view landing page
- view course previews
- click snippet preview
- view pricing

Cannot:

- access short trial lesson
- access full course
- download paid resources
- receive certificates

### Paid Level Test user

Can:

- take Level Test
- receive score report
- receive diagnostic certificate
- get mapped pathway
- unlock short trial lesson

### Paid course user

Can:

- access purchased course
- complete lessons and activities
- download course worksheets
- complete final assessment
- receive course completion certificate

### Subscription user

Can:

- access all self-study courses while subscription is active
- receive completion certificates for completed courses
- lose access if subscription fails or is cancelled

### Institution user

Can:

- access assigned courses while institution licence is active
- appear in institution dashboard and reports

## Refund rules

- Level Test: refund only before the test is started.
- Single course: 14-day refund if less than 20% completed and no certificate issued.
- Bundle: 14-day refund if no more than one course started and no certificate issued.
- Subscription: cancellation stops future billing. First billing refund only if usage is minimal.
- Institution licence: subject to written agreement and onboarding status.
