# Developer Deployment Checklist
## Before synthesis
- [ ] AWS account ready.
- [ ] Polly access enabled.
- [ ] S3 bucket created.
- [ ] AWS profile configured.
- [ ] Region selected.
- [ ] Voice availability checked.
- [ ] SSML validation passed.

## Synthesis
- [ ] Run sample batch.
- [ ] QA first five files.
- [ ] Generate full batch.
- [ ] Upload full batch to S3.
- [ ] Import media assets to LMS.
- [ ] Attach media to lessons and activities.

## Post-synthesis
- [ ] Human audio QA completed.
- [ ] Captions reviewed and timed.
- [ ] LMS audio player tested.
- [ ] Learner access rules tested.
