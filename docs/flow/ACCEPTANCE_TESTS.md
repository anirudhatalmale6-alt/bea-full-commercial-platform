# Mandatory Acceptance Tests

1. Public preview → Level Test checkout.
2. Level Test payment → Level Test unlock.
3. Placement completion → CEFR mapped pathway, diagnostic report and certificate.
4. Short trial unlock → only after paid/completed Level Test and only for mapped course.
5. Full course checkout → payment webhook → enrolment created → dashboard unlock.
6. LMS Module 1 opens and lesson progress saves.
7. Parent dashboard updates for linked child only.
8. Teacher dashboard shows assigned class progress.
9. Institution dashboard shows licence, seats, learners, teachers and reports.
