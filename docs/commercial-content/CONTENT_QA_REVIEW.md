# BEA Content QA Review

A lesson is commercially ready only when all checks pass.

## Lesson-level QA

- [ ] Learner-facing explanation is clear.
- [ ] Teaching input is complete.
- [ ] Examples are level-appropriate.
- [ ] Interactive practice exists.
- [ ] Correct answers are included.
- [ ] Distractors are plausible.
- [ ] Instant feedback is included.
- [ ] Teacher notes are included.
- [ ] Homework task is included.
- [ ] Assessment task is included.
- [ ] Speaking task is included.
- [ ] Writing task is included.
- [ ] Listening script or audio prompt is included.
- [ ] Visual-card manifest is included.
- [ ] Worksheet markdown exists.
- [ ] Answer-key markdown exists.
- [ ] Completion rule is defined.
- [ ] Accessibility check complete.
- [ ] No trademarked third-party content copied.

## Module-level QA

- [ ] Six lessons are present.
- [ ] Module outcomes are present.
- [ ] Module assessment exists.
- [ ] Skills are balanced.
- [ ] Vocabulary and grammar progression is logical.

## Course-level QA

- [ ] Six modules are present.
- [ ] Final level assessment exists.
- [ ] Course completion rule exists.
- [ ] Certificate rule exists.
- [ ] Learner workbook exists or can be generated.
- [ ] Teacher pack exists or can be generated.
