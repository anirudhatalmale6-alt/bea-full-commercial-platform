# Final Level Assessment Specification

Each CEFR course must end with a final level assessment.

## Required sections

1. Use of English
2. Reading
3. Listening
4. Speaking
5. Writing

## Required scoring

- Total score: 100
- Pass score: 70
- Distinction: 85
- Speaking and writing require moderation
- Final certificate requires final assessment pass and course progress completion

## Required output

When passed, the platform should:

1. update course_enrollments.completed_at
2. generate course completion report
3. issue course completion certificate
4. unlock next-level course recommendation
5. email learner with certificate and next steps
