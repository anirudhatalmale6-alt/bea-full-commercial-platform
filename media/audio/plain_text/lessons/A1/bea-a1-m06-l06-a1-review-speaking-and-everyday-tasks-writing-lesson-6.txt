Learning objective: Learners can name, identify, ask and answer basic questions in a writing task about food.

Target language: basic imperatives.
Explanation.
In this lesson you will practise writing for a1 review, speaking and everyday tasks. You will learn useful language connected to food and practise basic imperatives. By the end, you should be able to complete a short real-life task and explain your answer clearly at A1 level.
Model examples.

Example 2: The learner gives a A1 answer with clear meaning and correct key language.
Example 3: The learner checks, improves and repeats the answer after feedback.

Question.

Option A. A clear answer using the target language.
Option B. An answer with missing information.
Option C. An answer that does not match the situation.

Speaking practice.

Writing practice.
