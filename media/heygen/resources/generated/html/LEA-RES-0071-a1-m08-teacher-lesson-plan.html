A1-M08 Teacher lesson plan: Stories, literature and creative expression

British English Academy - British English - Global Learning

Resource identity

Course: Starter English
Level: A1
Module: A1-M08 - Stories, literature and creative expression
Resource type: Teacher lesson plan
Adaptive use: core/remediation/extension depending on score
Learner expectation: short phrases, simple sentences, clear model answers and picture-supported tasks.

Learning outcomes

  • By the end of this module, learners can listen for key words using like + noun/verb-ing within the theme of Stories, literature and creative expression.
  • Learners can understand and use key vocabulary: genres, characters, settings, plot, figurative language, stories, literature, creative.
  • Learners complete a spoken or written product that can be scored against the CEFR skill focus.

Five-lesson sequence

  • Lesson 1: Activate prior knowledge, introduce model language, check meaning with visuals.
  • Lesson 2: Controlled grammar and vocabulary practice with pair checking.
  • Lesson 3: Reading/listening input with gist and detail questions.
  • Lesson 4: Speaking/writing production using frames and feedback.
  • Lesson 5: Project, review quiz, reflection, and adaptive next step.

Teacher script

Today we are learning how to listen for key words. First, listen to the model. Then practise with a partner. Use the word bank and sentence frame. I will check accuracy, confidence and independence. Mistakes are evidence for our next step, not failure.

Differentiation and adaptation

  • Support: reduce choices, provide sentence starters, replay listening, model one example.
  • Core: complete the main task with peer rehearsal and teacher feedback.
  • Extension: add reasons, examples, comparison, register control or longer output depending on level.

Assessment evidence

Use auto-marked quiz + teacher-scored speaking/writing task. Record accuracy, fluency, range, interaction and task completion. Route learners to remediation if score is below 70%; unlock extension above 90% with low prompt support.
