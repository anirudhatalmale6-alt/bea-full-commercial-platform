## Purpose
This progress report supports pre-k progress report through short, practical, child-friendly learning routines.

## How to use
- 1. Model the routine with one example.
- 2. Let the learner try with support.
- 3. Give immediate praise and one correction.
- 4. Record accuracy, prompt level and confidence.
- 5. Repeat or extend based on evidence.

- Card 2: words

## Assessment note
Mastery is shown when the learner responds correctly with low prompt support across two opportunities.
