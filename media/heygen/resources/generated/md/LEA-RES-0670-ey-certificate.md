# Grade 3 certificate

## Resource identity
Course: Structured Literacy
Level: Grade 3
Module: EY
Resource type: certificate
Adaptive use: core/intervention/parent support

## Purpose
This certificate supports Grade 3 structured literacy through short, practical, child-friendly learning routines.

## How to use
- Model one example.
- Let the learner try with support.
- Give immediate praise and one correction.
- Record accuracy, prompt level and confidence.
- Repeat or extend based on evidence.

## Ready-made content
- Card 1: read
- Card 2: explain
- Card 3: evidence
- Card 4: improve

## Assessment note
Mastery is shown when the learner responds correctly with low prompt support across two opportunities.
