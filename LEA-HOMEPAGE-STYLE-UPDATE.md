# British English Academy Homepage Style Update

## What changed

The homepage has been redesigned around a course-marketplace landing page architecture:

- top information strip
- sticky navigation
- large hero section
- course navigator/search module
- quick learning-goal pills
- paid placement-test conversion card
- popular skill areas
- popular course cards
- CEFR course levels
- placement-test workflow
- institutional footer

## Colour scheme

The platform now uses a Pearson-inspired colour direction:

- Deep purple: `#361163`
- Plum: `#52237f`
- Teal: `#00a7a3`
- Mint: `#dff8f6`
- Yellow: `#ffb81c`
- Coral: `#ff6955`
- Soft lilac: `#efe7f7`

## Commercial pathway

The homepage keeps the paid placement test as the main conversion route:

1. Browse English courses
2. Use the course navigator
3. Take the paid CEFR placement test
4. Receive score report and diagnostic certificate
5. Join the recommended paid course pathway

## Compliance note

This is an original British English Academy design inspired by public-facing layout and colour-direction benchmarks. It does not copy third-party source code, imagery or proprietary course content.
