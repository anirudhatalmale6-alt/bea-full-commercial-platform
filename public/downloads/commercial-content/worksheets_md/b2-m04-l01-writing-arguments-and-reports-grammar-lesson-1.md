# British English Academy Worksheet
## B2-M04-L01 - Writing Arguments and Reports: Grammar Lesson 1

**CEFR level:** B2  
**Skill:** grammar  
**Objective:** Learners can argue, evaluate, clarify and organise detailed communication in a grammar task about global issues.

### 1. Learn
In this lesson you will practise grammar for writing arguments and reports. You will learn useful language connected to global issues and practise participle clauses. By the end, you should be able to complete a short real-life task and explain your answer clearly at B2 level.

### 2. Examples
- Example 1: I can use participle clauses to talk about global issues.
- Example 2: The learner gives a B2 answer with clear meaning and correct key language.
- Example 3: The learner checks, improves and repeats the answer after feedback.
- Example 4: The learner uses one linking phrase or support phrase suitable for B2.

### 3. Practice
1. Choose the best sentence using participle clauses.
2. Match the global issues phrase to its meaning.
3. Build a complete answer for the lesson task.

### 4. Speaking Task
Speak for 60-120 seconds about global issues. Use at least two examples and one reason.

### 5. Writing Task
Write a short B2 response about global issues. Include the target language: participle clauses.

### 6. Homework
Write or record five sentences about global issues using language from this lesson.
