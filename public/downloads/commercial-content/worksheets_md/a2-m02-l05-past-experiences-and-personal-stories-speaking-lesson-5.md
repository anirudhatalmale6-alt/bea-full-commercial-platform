# British English Academy Worksheet
## A2-M02-L05 - Past Experiences and Personal Stories: Speaking Lesson 5

**CEFR level:** A2  
**Skill:** speaking  
**Objective:** Learners can describe, compare and exchange simple information in a speaking task about weather.

### 1. Learn
In this lesson you will practise speaking for past experiences and personal stories. You will learn useful language connected to weather and practise basic modals. By the end, you should be able to complete a short real-life task and explain your answer clearly at A2 level.

### 2. Examples
- Example 1: I can use basic modals to talk about weather.
- Example 2: The learner gives a A2 answer with clear meaning and correct key language.
- Example 3: The learner checks, improves and repeats the answer after feedback.
- Example 4: The learner uses one linking phrase or support phrase suitable for A2.

### 3. Practice
1. Choose the best sentence using basic modals.
2. Match the weather phrase to its meaning.
3. Build a complete answer for the lesson task.

### 4. Speaking Task
Speak for 60-120 seconds about weather. Use at least two examples and one reason.

### 5. Writing Task
Write a short A2 response about weather. Include the target language: basic modals.

### 6. Homework
Write or record five sentences about weather using language from this lesson.
