# British English Academy Worksheet
## A1-M05-L06 - Daily Routines and Free Time: Writing Lesson 6

**CEFR level:** A1  
**Skill:** writing  
**Objective:** Learners can name, identify, ask and answer basic questions in a writing task about classroom.

### 1. Learn
In this lesson you will practise writing for daily routines and free time. You will learn useful language connected to classroom and practise can/can't. By the end, you should be able to complete a short real-life task and explain your answer clearly at A1 level.

### 2. Examples
- Example 1: I can use can/can't to talk about classroom.
- Example 2: The learner gives a A1 answer with clear meaning and correct key language.
- Example 3: The learner checks, improves and repeats the answer after feedback.
- Example 4: The learner uses one linking phrase or support phrase suitable for A1.

### 3. Practice
1. Choose the best sentence using can/can't.
2. Match the classroom phrase to its meaning.
3. Build a complete answer for the lesson task.

### 4. Speaking Task
Speak for 60-120 seconds about classroom. Use at least two examples and one reason.

### 5. Writing Task
Write a short A1 response about classroom. Include the target language: can/can't.

### 6. Homework
Write or record five sentences about classroom using language from this lesson.
