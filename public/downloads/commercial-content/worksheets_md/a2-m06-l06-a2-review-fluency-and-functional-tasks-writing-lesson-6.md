# British English Academy Worksheet
## A2-M06-L06 - A2 Review, Fluency and Functional Tasks: Writing Lesson 6

**CEFR level:** A2  
**Skill:** writing  
**Objective:** Learners can describe, compare and exchange simple information in a writing task about free time.

### 1. Learn
In this lesson you will practise writing for a2 review, fluency and functional tasks. You will learn useful language connected to free time and practise going to. By the end, you should be able to complete a short real-life task and explain your answer clearly at A2 level.

### 2. Examples
- Example 1: I can use going to to talk about free time.
- Example 2: The learner gives a A2 answer with clear meaning and correct key language.
- Example 3: The learner checks, improves and repeats the answer after feedback.
- Example 4: The learner uses one linking phrase or support phrase suitable for A2.

### 3. Practice
1. Choose the best sentence using going to.
2. Match the free time phrase to its meaning.
3. Build a complete answer for the lesson task.

### 4. Speaking Task
Speak for 60-120 seconds about free time. Use at least two examples and one reason.

### 5. Writing Task
Write a short A2 response about free time. Include the target language: going to.

### 6. Homework
Write or record five sentences about free time using language from this lesson.
