# British English Academy Worksheet
## C1-M01-L05 - Advanced Academic and Professional Discourse: Speaking Lesson 5

**CEFR level:** C1  
**Skill:** speaking  
**Objective:** Learners can synthesise, justify, negotiate and adapt register in a speaking task about analysis.

### 1. Learn
In this lesson you will practise speaking for advanced academic and professional discourse. You will learn useful language connected to analysis and practise hedging. By the end, you should be able to complete a short real-life task and explain your answer clearly at C1 level.

### 2. Examples
- Example 1: I can use hedging to talk about analysis.
- Example 2: The learner gives a C1 answer with clear meaning and correct key language.
- Example 3: The learner checks, improves and repeats the answer after feedback.
- Example 4: The learner uses one linking phrase or support phrase suitable for C1.

### 3. Practice
1. Choose the best sentence using hedging.
2. Match the analysis phrase to its meaning.
3. Build a complete answer for the lesson task.

### 4. Speaking Task
Speak for 60-120 seconds about analysis. Use at least two examples and one reason.

### 5. Writing Task
Write a short C1 response about analysis. Include the target language: hedging.

### 6. Homework
Write or record five sentences about analysis using language from this lesson.
