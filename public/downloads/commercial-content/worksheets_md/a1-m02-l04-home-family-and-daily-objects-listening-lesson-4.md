# British English Academy Worksheet
## A1-M02-L04 - Home, Family and Daily Objects: Listening Lesson 4

**CEFR level:** A1  
**Skill:** listening  
**Objective:** Learners can name, identify, ask and answer basic questions in a listening task about food.

### 1. Learn
In this lesson you will practise listening for home, family and daily objects. You will learn useful language connected to food and practise basic imperatives. By the end, you should be able to complete a short real-life task and explain your answer clearly at A1 level.

### 2. Examples
- Example 1: I can use basic imperatives to talk about food.
- Example 2: The learner gives a A1 answer with clear meaning and correct key language.
- Example 3: The learner checks, improves and repeats the answer after feedback.
- Example 4: The learner uses one linking phrase or support phrase suitable for A1.

### 3. Practice
1. Choose the best sentence using basic imperatives.
2. Match the food phrase to its meaning.
3. Build a complete answer for the lesson task.

### 4. Speaking Task
Speak for 60-120 seconds about food. Use at least two examples and one reason.

### 5. Writing Task
Write a short A1 response about food. Include the target language: basic imperatives.

### 6. Homework
Write or record five sentences about food using language from this lesson.
