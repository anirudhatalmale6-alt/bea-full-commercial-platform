# British English Academy Worksheet
## A2-M06-L04 - A2 Review, Fluency and Functional Tasks: Listening Lesson 4

**CEFR level:** A2  
**Skill:** listening  
**Objective:** Learners can describe, compare and exchange simple information in a listening task about health.

### 1. Learn
In this lesson you will practise listening for a2 review, fluency and functional tasks. You will learn useful language connected to health and practise comparatives. By the end, you should be able to complete a short real-life task and explain your answer clearly at A2 level.

### 2. Examples
- Example 1: I can use comparatives to talk about health.
- Example 2: The learner gives a A2 answer with clear meaning and correct key language.
- Example 3: The learner checks, improves and repeats the answer after feedback.
- Example 4: The learner uses one linking phrase or support phrase suitable for A2.

### 3. Practice
1. Choose the best sentence using comparatives.
2. Match the health phrase to its meaning.
3. Build a complete answer for the lesson task.

### 4. Speaking Task
Speak for 60-120 seconds about health. Use at least two examples and one reason.

### 5. Writing Task
Write a short A2 response about health. Include the target language: comparatives.

### 6. Homework
Write or record five sentences about health using language from this lesson.
