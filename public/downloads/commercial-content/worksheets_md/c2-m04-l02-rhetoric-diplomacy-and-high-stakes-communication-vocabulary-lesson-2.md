# British English Academy Worksheet
## C2-M04-L02 - Rhetoric, Diplomacy and High-Stakes Communication: Vocabulary Lesson 2

**CEFR level:** C2  
**Skill:** vocabulary  
**Objective:** Learners can critique, refine, interpret nuance and produce expert-level responses in a vocabulary task about abstract argument.

### 1. Learn
In this lesson you will practise vocabulary for rhetoric, diplomacy and high-stakes communication. You will learn useful language connected to abstract argument and practise register control. By the end, you should be able to complete a short real-life task and explain your answer clearly at C2 level.

### 2. Examples
- Example 1: I can use register control to talk about abstract argument.
- Example 2: The learner gives a C2 answer with clear meaning and correct key language.
- Example 3: The learner checks, improves and repeats the answer after feedback.
- Example 4: The learner uses one linking phrase or support phrase suitable for C2.

### 3. Practice
1. Choose the best sentence using register control.
2. Match the abstract argument phrase to its meaning.
3. Build a complete answer for the lesson task.

### 4. Speaking Task
Speak for 60-120 seconds about abstract argument. Use at least two examples and one reason.

### 5. Writing Task
Write a short C2 response about abstract argument. Include the target language: register control.

### 6. Homework
Write or record five sentences about abstract argument using language from this lesson.
