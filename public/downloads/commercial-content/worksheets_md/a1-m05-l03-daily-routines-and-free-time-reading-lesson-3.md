# British English Academy Worksheet
## A1-M05-L03 - Daily Routines and Free Time: Reading Lesson 3

**CEFR level:** A1  
**Skill:** reading  
**Objective:** Learners can name, identify, ask and answer basic questions in a reading task about personal information.

### 1. Learn
In this lesson you will practise reading for daily routines and free time. You will learn useful language connected to personal information and practise be: am/is/are. By the end, you should be able to complete a short real-life task and explain your answer clearly at A1 level.

### 2. Examples
- Example 1: I can use be: am/is/are to talk about personal information.
- Example 2: The learner gives a A1 answer with clear meaning and correct key language.
- Example 3: The learner checks, improves and repeats the answer after feedback.
- Example 4: The learner uses one linking phrase or support phrase suitable for A1.

### 3. Practice
1. Choose the best sentence using be: am/is/are.
2. Match the personal information phrase to its meaning.
3. Build a complete answer for the lesson task.

### 4. Speaking Task
Speak for 60-120 seconds about personal information. Use at least two examples and one reason.

### 5. Writing Task
Write a short A1 response about personal information. Include the target language: be: am/is/are.

### 6. Homework
Write or record five sentences about personal information using language from this lesson.
